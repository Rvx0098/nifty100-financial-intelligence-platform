# Deployment

## Clone
- git clone <repository-url>
- cd n100-financial-intelligence

## Virtual Environment
- python -m venv venv
- .\venv\Scripts\Activate.ps1

## Install
- pip install -r requirements.txt

## Run
- python -m src.visualization.export_excel

## Generate Reports
- python -m src.visualization.export_excel

## Run Tests
- pytest
